Admin editor

Manage the editable site without touching JSON by hand.

This panel edits the public pages at /editable/ and /editable/behindtheshadow/. It does not write directly to GitHub Pages, so the final step is still to upload the exported files to GitHub.

This is an admin tool by convention, not by real authentication. On a static host like GitHub Pages, anyone with the URL can open it.

Recommended GitHub flow

1. Download the files from this panel.
2. Open the repository on GitHub.
3. Replace the matching files under projects/moretti/editable/….
4. Commit the change.

Local workflow

1. Run npm start.
2. Open http://localhost:4173/editable/admin/.
3. Click Save to disk.
4. Commit and push from the repository.
