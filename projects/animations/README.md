# Tongatron Animations Lab

Demo e libreria locale per sperimentare animazioni riusabili:

- `index.html`: demo principale
- `animations.js`: runtime che collega trigger, toggle e stato
- `animations.config.js`: definizioni e default degli effetti
- `animations.css`: stili comuni degli effetti
- `effects/`: un file per effetto

Effetti attuali:

- balloons
- glitch
- word-swap
- falling-blocks
- glass-crack

Obiettivo:

- testare più effetti insieme
- rendere ogni animazione esportabile su altre pagine
- tenere separati trigger, parametri, runtime e demo
