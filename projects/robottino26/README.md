# ROBOTTINO - testa rotante interattiva

Repository del progetto Robottino (hardware, design e libreria Arduino).

## Contenuti
- `Software/Robottino-ArduinoLibrary`: libreria Arduino + esempi `.ino`
- `Hardware`: schemi e board PCB (`.sch`, `.brd`, `.pdf`)
- `Design`: file `.dxf` del case
- `Education`: materiali didattici

## Avvio rapido (software)
Consulta la guida completa in `Software/Robottino-ArduinoLibrary/README.md`.
