THINGS YOU NEED


ARDUINO BOARD

Robottino is compatible with Arduino boards: Uno, Leonardo, Yùn (web connection).


COMPONENTS

cost: 30-35 $

  • OLED Display 128 x 64 (view)
  • Ultrasonic Ranging Module
    HC-SR04 (view)
  • Analog 180° Micro Servo module (view)
  • Diffuse LED rgb 10 mm (view)
  • Small Buzzer 5V (view)
  • 2 x Small photoresistors (view)
  • Pin Female Header, with extra long legs (view)
    (2 x 8 ways, 1 x 6 ways, 1 x 10 ways)
  • Resistors: 3 x 200 Ohm resistors, 2 x 1K Ohm
  • 1 Capacitor 100 uf


TOOLS

  • Soldering iron and flux
  • Wires
  • Heat-shrink tubing
  • Hot-glue gun

SCHEMATICS AND DESIGN

The breadboard file and the circuit scheme will guide you through the wiring of electronics components.
Use the design file (.eps) to make the case, you need an MDF panel with 4mm thickness and access to a lasercut machine.
