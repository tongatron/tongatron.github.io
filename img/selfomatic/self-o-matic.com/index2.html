Self-O-Matic

Social camera

Aggiorna in diretta l’album della tua pagina Facebook

Gli utenti generano gli autoscatti premendo il pulsante. La camera scatta la fotografia e la pubblica istantaneamente sui social network e sul web: album facebook, twitter streaming, siti internet.

Noleggia per il tuo evento

COME FUNZIONA

Autoscatti
Gli utenti che interagiscono con l'installazione premono il pulsante per generare l'austoscatto.

Pubblicazione immediata
La fotografia viene istantaneamente pubblicata sull’album fotografico della tua pagina facebook, generando una diretta delle immagini scattate dagli ospiti dell'evento.

SOLUZIONI A NOLEGGIO

Installazione
Installiamo le macchinette presso la location del tuo evento. Presidiamo per eventi di breve durata e realizziamo anche installazioni autonome dalla presenza di un operatore.

Setting
Impostiamo il software per pubblicare sulla pagina del tuo evento con le modalità di pubblicazione personalizzate: watermark sulle immagini, pubblicazione con #hashtag, identificazione utenti con app smartphone.

PIATTAFORMA

Custom design
La soluzione standard è proposta con il design “Big Camera”, ma abbiamo pieno controllo sul sistema costruttivo per poterlo adattare a qualsiasi esigenza estetica o funzionale. Scopri i design.

Piattaforma tecnologica
Il nostro software, sviluppato in Python e rilasciato con licenza open source, è integrato con le API Facebook e può integrarsi con Twitter, CMS e molte altre piattaforme web. Utilizziamo la piattaforma Raspberry Pi, un computer Linux low-cost delle dimensioni di una carta di credito.

Il progetto

Il progetto è sviluppato da Giovanni Bindi, Fabrizio Guglielmino e Giacomo Leonzi, tre maker di estrazione differente accumunati dalla passione per lo sviluppo di oggetti digitali. Adottiamo tecnologie open source e realizziamo sistemi fisici digitali, sviluppiamo presso il laboratorio di manifattura digitale FabLab Torino.

DESIGN
Abbiamo pieno controllo sul sistema costruttivo per poterlo adattare a qualsiasi esigenza estetica o funzionale. Scopri i design.

HARDWARE
Utilizziamo la piattaforma Raspberry Pi, un computer Linux low-cost delle dimensioni di una carta di credito.

SOFTWARE
Il software Self-O-Matic, sviluppato in Python e rilasciato con licenza open source, è integrato con le Facebook Graph API per la pubblicazione dei contenuti. Il sistema è configurabile da remoto e in grado di inviare agli amministratori notifiche push con Pushetta.
Sviluppiamo interazioni personalizzate e integrazioni con Twitter, CMS e altre piattaforme web.

CONTATTACI

Noleggiamo la soluzione standard e realizziamo installazioni su misura.
Per eventi, manifestazioni, feste, locali e social media marketing.
